#  Advanced Calculator

A modern, feature-rich, and responsive calculator built with **HTML, CSS, and JavaScript**. This project goes beyond basic arithmetic by offering an elegant UI, calculation history, theme switching, keyboard support, and scientific functionalities.

##  Features

*  Basic Arithmetic Operations

  * Addition
  * Subtraction
  * Multiplication
  * Division

*  Dark & Light Theme Toggle

*  Calculation History

  * Stores previous calculations
  * Easy access to recent operations

*  Keyboard Support

  * Perform calculations directly from the keyboard

*  Scientific Calculator Functions

  * Advanced mathematical operations
  * Enhanced user experience

*  Fully Responsive Design

  * Works smoothly on desktop, tablet, and mobile devices

*  Modern User Interface

  * Clean design
  * Smooth animations
  * Interactive button effects

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome Icons

##  Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/advanced-calculator.git
```

### 2. Navigate to the Project Folder

```bash
cd advanced-calculator
```

### 3. Open the Project

Simply open `index.html` in your browser.
